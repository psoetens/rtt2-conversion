
Please run the rtt2-converter tool *after* you have run the 'to-rtt-2.0.pl script !

See http://www.orocos.org/wiki/rtt/rtt-20/rtt-20-renaming-table for the latest updates
and instructions.


If you don't have boost 1.41 or newer, unpack the rtt2-converter-boost.tar.bz2 package too in the src/
directory:

tar xzf rtt2-converter.tar.gz
cd rtt2-converter/src
tar xjf rtt2-converter-boost.tar.bz2
cd ..
make

And then follow the instructions on the above link.
